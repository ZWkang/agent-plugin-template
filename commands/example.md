# /example Command

This is an example command. When invoked, it demonstrates the basic command structure.

## Usage

```
/example [options]
```

## What This Command Does

1. Displays a greeting message
2. Shows the current working directory
3. Lists available options

## Options

- `--verbose`: Show detailed output
- `--help`: Display help information

## Example Output

```
Hello from /example command!
Working directory: /path/to/project
Available options: --verbose, --help
```
